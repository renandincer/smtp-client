# stmp-client
A first timer experiment with Python.

## Usage
1. Go to the directory you want to try this at
2. `git clone git@github.com:renandincer/stmp-client.git`
3. `cd stmp-client`
4. `pyhton smtp.py`
5. follow on screen instructions.
